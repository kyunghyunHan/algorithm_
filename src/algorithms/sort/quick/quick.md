# Quick Sort

### Best-case time complexity
#### )Recurrence Equation
- T(n) = ⎰ Θ(1),  n=1            ⎱
         ⎱ 2T(n/2) + Θ(n), n ≥ 2 ⎰
#### )Closed Type
- T(n) = Θ(n log n)

### Worst-case time complexity
#### )Recurrence Equation
- T(n) = ⎰ Θ(1),  n=1            ⎱
         ⎱ T(n-1) + Θ(n), n ≥ 2  ⎰
#### )Closed Type
- T(n) = Θ(n²)

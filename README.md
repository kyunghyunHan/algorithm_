# 확률론의 기초와 응용

## 확률
- 확률은 어떤 일이 일어날 가능성을 0과 1사이의 수치로 표현한것입니다.어떤일이 일어날 가능성이 없다면
그 확률은 0이 될 것이고 반드시 일어난다면 1이 될것입니다.하지만 일상에서는 퍼센트로 얘기합니다.예를 들어
0.25면 25%를 뜻하게 되는 것입니다.

## 1. 표본공간과 사건

확률론을 시작하기 전에 가장 기본이 되는 개념인 표본공간과 사건에 대해 알아보겠습니다.

**표본공간(Sample Space)**은 실험이나 관찰에서 발생할 수 있는 모든 가능한 결과의 집합입니다. 보통 기호 Ω(오메가)로 표시합니다. 표본공간의 각 원소를 표본점(sample point)이라고 부릅니다.

예를 들어, 주사위를 한 번 던지는 실험에서 표본공간은 Ω = {1, 2, 3, 4, 5, 6}이 됩니다. 동전을 한 번 던지는 실험에서는 Ω = {앞면, 뒷면}입니다.

**사건(Event)**은 표본공간의 부분집합으로, 우리가 관심을 갖는 결과들의 모임입니다. 사건은 대문자 알파벳(A, B, C 등)으로 표기합니다.

주사위 예시에서, "짝수가 나오는 사건" A = {2, 4, 6}, "3보다 큰 수가 나오는 사건" B = {4, 5, 6}과 같이 정의할 수 있습니다.

사건들 사이에는 집합 연산이 가능합니다:
- **합집합(Union)**: A∪B는 사건 A 또는 사건 B가 일어나는 경우입니다.
- **교집합(Intersection)**: A∩B는 사건 A와 사건 B가 동시에 일어나는 경우입니다.
- **여집합(Complement)**: A^c는 사건 A가 일어나지 않는 경우입니다.

주사위 예시에서 A∪B = {2, 4, 5, 6}, A∩B = {4, 6}, A^c = {1, 3, 5}입니다.

**특수한 사건**:
- **전체사건(Sure Event)**: 표본공간 전체 Ω는 항상 일어나는 사건입니다.
- **공사건(Impossible Event)**: 공집합 ∅는 절대 일어나지 않는 사건입니다.
- **서로소 사건(Mutually Exclusive Events)**: A∩B = ∅인 경우, A와 B는 동시에 일어날 수 없는 서로소 사건입니다.

사건의 개념은 확률 계산의 기초가 되며, 복잡한 확률 문제를 해결할 때 사건 간의 관계를 이해하는 것이 중요합니다.

## 2. 확률변수

- 동전 하나 던지기의 결과를 x라는 변수로 표기하기로 한다면 이 x를 확률변수라고 부릅니다.

확률변수는 보통 대문자(X, Y, Z 등)로 표기하며, 그 값은 소문자(x, y, z 등)로 표기합니다.

**이산확률변수(Discrete Random Variable)**는 유한하거나 셀 수 있는 값들만 가질 수 있는 확률변수입니다.
예: 주사위의 눈금, 동전 던지기의 앞면 수, 가족의 자녀 수

**연속확률변수(Continuous Random Variable)**는 실수 구간 내의 무한한 값을 가질 수 있는 확률변수입니다.
예: 키, 몸무게, 시간, 온도

확률변수의 분포를 서술하는 방법에는 두 가지가 있습니다:

1. **확률질량함수(Probability Mass Function, PMF)**: 이산확률변수 X에 대해, P(X = x)는 X가 값 x를 가질 확률입니다.

2. **확률밀도함수(Probability Density Function, PDF)**: 연속확률변수 X에 대해, f(x)는 X가 x 근처의 값을 가질 확률의 밀도를 나타냅니다. 특정 구간 [a, b]에서의 확률은 ∫[a,b] f(x)dx로 계산됩니다.

확률변수의 중요한 특성으로는 다음이 있습니다:

- **기대값(Expectation)**: E(X) = Σ x·P(X = x) (이산) 또는 E(X) = ∫ x·f(x)dx (연속)
  기대값은 확률변수의 '평균적인 값'을 나타냅니다.

- **분산(Variance)**: Var(X) = E[(X - E(X))²] = E(X²) - [E(X)]²
  분산은 확률변수 값이 기대값으로부터 얼마나 퍼져있는지를 나타냅니다.

- **표준편차(Standard Deviation)**: σ = √Var(X)
  표준편차는 분산의 제곱근으로, 기대값으로부터의 평균적인 편차를 원래 단위로 나타냅니다.

확률변수는 확률론과 통계학의 핵심 개념으로, 실제 데이터 분석과 모델링에 광범위하게 사용됩니다.

## 3. 단일사건의 확률

확률은 사건이 일어날 가능성을 0에서 1 사이의 숫자로 나타낸 것입니다. 단일 사건 A의 확률은 P(A)로 표기합니다.

확률은 다음과 같은 공리(Axioms)를 만족합니다:

1. **비음수성(Non-negativity)**: 모든 사건 A에 대해, P(A) ≥ 0
2. **정규화(Normalization)**: 표본공간 Ω에 대해, P(Ω) = 1
3. **가산가법성(Countable Additivity)**: 서로소인 사건들 A₁, A₂, ...에 대해, P(A₁∪A₂∪...) = P(A₁) + P(A₂) + ...

확률을 계산하는 접근법에는 여러 가지가 있습니다:

**고전적 확률(Classical Probability)**:
모든 기본 결과가 동일한 확률을 가질 때 사용합니다.
P(A) = (사건 A에 속하는 결과의 수) / (가능한 전체 결과의 수)

예: 공정한 주사위에서 짝수가 나올 확률
P(짝수) = |{2, 4, 6}| / |{1, 2, 3, 4, 5, 6}| = 3/6 = 1/2

**상대도수 확률(Relative Frequency Probability)**:
실험을 여러 번 반복했을 때 사건이 발생한 비율을 통해 확률을 추정합니다.
P(A) ≈ (사건 A가 발생한 횟수) / (전체 실험 횟수)

**주관적 확률(Subjective Probability)**:
개인의 믿음이나 판단에 기반한 확률로, 베이지안 접근법에서 중요합니다.

단일 사건의 확률을 계산할 때는 다음 성질도 유용합니다:

- **여사건의 확률**: P(A^c) = 1 - P(A)
- **확실한 사건의 확률**: P(Ω) = 1
- **불가능한 사건의 확률**: P(∅) = 0
- **확률의 범위**: 모든 사건 A에 대해, 0 ≤ P(A) ≤ 1

실생활에서 확률은 날씨 예보, 보험 계산, 의학적 진단, 도박 등 다양한 분야에서 불확실성을 정량화하는 데 사용됩니다.

## 4. 합의 법칙

합의 법칙(Addition Rule)은 두 사건의 합집합에 대한 확률을 계산하는 방법을 제공합니다. 

**일반적인 합의 법칙**:
두 사건 A와 B에 대해,
P(A∪B) = P(A) + P(B) - P(A∩B)

이 공식은 "A 또는 B가 일어날 확률"을 계산합니다. P(A) + P(B)를 그냥 더하면 A와 B가 동시에 일어나는 경우(A∩B)가 두 번 계산되므로, 한 번 빼주어야 합니다.

예: 한 학생이 수학 시험을 통과할 확률이 0.7, 영어 시험을 통과할 확률이 0.6, 두 시험 모두 통과할 확률이 0.4라면, 적어도 한 과목을 통과할 확률은?
P(수학∪영어) = P(수학) + P(영어) - P(수학∩영어) = 0.7 + 0.6 - 0.4 = 0.9

**서로소 사건에 대한 합의 법칙**:
만약 A와 B가 서로소 사건(A∩B = ∅)이라면, 간단히
P(A∪B) = P(A) + P(B)

예: 주사위를 던져서 1이 나오거나 짝수가 나올 확률은?
A = {1}, B = {2, 4, 6}이라 하면, A∩B = ∅
P(A∪B) = P(A) + P(B) = 1/6 + 3/6 = 4/6 = 2/3

**세 개 이상 사건에 대한 확장**:
세 사건 A, B, C에 대해,
P(A∪B∪C) = P(A) + P(B) + P(C) - P(A∩B) - P(A∩C) - P(B∩C) + P(A∩B∩C)

이것은 포함-배제 원리(Inclusion-Exclusion Principle)의 일부로, 더 많은 사건으로 확장할 수 있습니다.

**응용 전략**:
1. 문제를 분석하여 관련 사건들을 명확히 정의합니다.
2. 각 사건의 확률과 교집합의 확률을 구합니다.
3. 적절한 합의 법칙 공식을 적용합니다.
4. 여사건을 이용하면 계산이 간단해지는 경우가 많습니다: P(A∪B) = 1 - P(A^c∩B^c)

합의 법칙은 "또는(OR)"으로 연결된 복합 사건의 확률을 계산할 때 필수적인 도구입니다.

## 5. 곱의 법칙

곱의 법칙(Multiplication Rule)은 두 사건의 교집합, 즉 두 사건이 동시에 일어날 확률을 계산하는 방법을 제공합니다.

**일반적인 곱의 법칙**:
두 사건 A와 B에 대해,
P(A∩B) = P(A) × P(B|A) = P(B) × P(A|B)

여기서 P(B|A)는 사건 A가 일어났다는 조건 하에 사건 B가 일어날 조건부 확률입니다.

예: 한 가방에 빨간 공 3개와 파란 공 2개가 있습니다. 두 개의 공을 비복원 추출할 때, 두 공 모두 빨간색일 확률은?
P(첫 번째가 빨강∩두 번째가 빨강) = P(첫 번째가 빨강) × P(두 번째가 빨강|첫 번째가 빨강) = (3/5) × (2/4) = 6/20 = 3/10

**독립 사건에 대한 곱의 법칙**:
두 사건 A와 B가 독립(independent)이라면, 한 사건의 발생이 다른 사건의 확률에 영향을 주지 않습니다. 이 경우,
P(A∩B) = P(A) × P(B)

예: 두 개의 주사위를 던질 때, 첫 번째 주사위에서 6이 나오고 두 번째 주사위에서 짝수가 나올 확률은?
두 주사위의 결과는 독립이므로,
P(첫 번째=6 ∩ 두 번째=짝수) = P(첫 번째=6) × P(두 번째=짝수) = (1/6) × (3/6) = 3/36 = 1/12

**독립성 판단**:
두 사건 A와 B가 독립인지 확인하려면, P(A∩B) = P(A) × P(B)가 성립하는지 검사하면 됩니다. 또는 P(B|A) = P(B) 또는 P(A|B) = P(A)가 성립하는지 확인할 수도 있습니다.

**순차적 사건의 확률**:
n개의 사건 A₁, A₂, ..., Aₙ에 대해,
P(A₁∩A₂∩...∩Aₙ) = P(A₁) × P(A₂|A₁) × P(A₃|A₁∩A₂) × ... × P(Aₙ|A₁∩A₂∩...∩Aₙ₋₁)

만약 모든 사건이 서로 독립이라면,
P(A₁∩A₂∩...∩Aₙ) = P(A₁) × P(A₂) × ... × P(Aₙ)

곱의 법칙은 "그리고(AND)"로 연결된 복합 사건의 확률을 계산할 때 사용되며, 특히 순차적인 실험이나 추출 문제에서 중요합니다.

## 6. 합의 법칙 보충

합의 법칙을 더 효과적으로 활용하기 위한 추가적인 개념과 기법들을 살펴보겠습니다.

**여사건을 이용한 확률 계산**:
때로는 직접 확률을 계산하는 것보다 여사건을 이용하는 것이 더 쉬울 수 있습니다.
P(A) = 1 - P(A^c)

예: 주사위를 두 번 던져서 적어도 한 번 6이 나올 확률은?
직접 계산: P(첫 번째=6 ∪ 두 번째=6)
여사건 이용: 1 - P(첫 번째≠6 ∩ 두 번째≠6) = 1 - (5/6)² = 1 - 25/36 = 11/36

**상호배타적 사건들의 합**:
사건 A₁, A₂, ..., Aₙ이 상호배타적(pairwise mutually exclusive)이면,
P(A₁∪A₂∪...∪Aₙ) = P(A₁) + P(A₂) + ... + P(Aₙ)

예: 주사위를 던져서 1, 3, 5가 나올 확률은?
P({1}∪{3}∪{5}) = P({1}) + P({3}) + P({5}) = 1/6 + 1/6 + 1/6 = 3/6 = 1/2

**드모르간 법칙(De Morgan's laws)**:
확률 계산에서도 집합론의 드모르간 법칙이 유용합니다.
(A∪B)^c = A^c ∩ B^c
(A∩B)^c = A^c ∪ B^c

예: 두 명의 학생 중 적어도 한 명이 시험에 합격하지 못할 확률을 구하려면,
P(A^c∪B^c) = 1 - P((A^c∪B^c)^c) = 1 - P(A∩B)
여기서 A와 B는 각각 학생들이 합격하는 사건입니다.

**전체 확률의 합은 1**:
표본공간을 완전히 분할하는 사건들의 확률 합은 1입니다.
P(A₁) + P(A₂) + ... + P(Aₙ) = 1 (단, A₁∪A₂∪...∪Aₙ = Ω 이고 Aᵢ∩Aⱼ = ∅, i≠j)

예: 주사위를 던질 때, P(1) + P(2) + P(3) + P(4) + P(5) + P(6) = 6 × (1/6) = 1

**복잡한 문제 해결 전략**:
1. 문제를 여러 간단한 사건으로 분해합니다.
2. 상호배타적 사건들로 재구성하면 계산이 간단해집니다.
3. 여사건과 드모르간 법칙을 활용합니다.
4. 확률을 계산하기 어려운 사건은 조건부 확률로 접근합니다.

합의 법칙의 적절한 활용은 복잡한 확률 문제를 해결하는 데 큰 도움이 됩니다.

## 7. 생일역설

**생일역설(Birthday Paradox)**은 직관에 반하는 확률 현상으로, 생각보다 적은 수의 사람들 중에서도 같은 생일을 가진 사람들이 존재할 확률이 높다는 것을 보여줍니다.

구체적으로는, 단 23명만 모여도 그중 두 사람 이상이 같은 생일을 가질 확률이 50%를 넘습니다. 이는 우리의 직관과 크게 다르기 때문에 '역설'이라고 불립니다.

**확률 계산**:
n명의 사람들 중에서 적어도 두 명이 같은 생일을 가질 확률을 계산해봅시다. 이를 직접 계산하기보다는 여사건을 이용하는 것이 편리합니다.

여사건 = '모든 사람의 생일이 서로 다르다'

n명의 사람이 모두 다른 생일을 가질 확률:
P(모두 다른 생일) = (365/365) × (364/365) × (363/365) × ... × ((365-n+1)/365)
                  = 365! / [(365-n)! × 365ⁿ]

따라서 적어도 두 명이 같은 생일을 가질 확률:
P(적어도 한 쌍이 같은 생일) = 1 - P(모두 다른 생일)

n = 23일 때, 이 확률은 약 0.5073으로, 50%를 넘습니다.
n = 50일 때는 약 0.97, n = 100일 때는 거의 1에 가깝습니다.

**직관에 반하는 이유**:
이 결과가 직관에 반하는 이유는, 우리가 보통 특정 생일에 대해 생각하기 때문입니다. 한 사람이 특정 날짜(예: 1월 1일)에 태어날 확률은 1/365로 낮습니다.

그러나 생일역설에서는 '아무 두 사람이나' 같은 생일을 가질 확률을 계산합니다. n명의 사람들 사이에는 n(n-1)/2쌍의 비교가 있습니다. 23명이라면 253쌍이 되어, 생각보다 많은 비교가 이루어집니다.

**일반화: 충돌 확률(Collision Probability)**:
생일역설은 더 일반적인 '충돌 확률' 문제의 한 예입니다. k개의 가능한 결과 중에서 n개의 무작위 선택을 할 때, 적어도 하나의 충돌(같은 선택)이 발생할 확률은 다음과 같습니다:
P(충돌) = 1 - k! / [(k-n)! × kⁿ]

**응용 분야**:
1. **암호학**: 해시 함수의 충돌 저항성(collision resistance) 분석
2. **데이터베이스**: 중복 키 발생 확률 추정
3. **네트워크 보안**: 랜덤 ID 생성 시 충돌 가능성 평가
4. **통계적 검정**: 무작위성 검정 및 패턴 탐지

생일역설은 확률적 직관을 기르는 중요한 사례이며, 직관만으로는 정확한 확률을 예측하기 어렵다는 점을 상기시켜 줍니다.

## 8. 조건부확률

**조건부확률(Conditional Probability)**은 어떤 사건이 이미 발생했다는 조건 하에서 다른 사건이 발생할 확률을 의미합니다. 사건 B가 일어났다는 조건 하에 사건 A가 일어날 확률은 P(A|B)로 표기합니다.

**정의**:
사건 B가 일어났다는 조건 하에 사건 A가 일어날 확률은 다음과 같이 정의됩니다:
P(A|B) = P(A∩B) / P(B), 단 P(B) > 0

이는 B라는 새로운 표본공간에서 A와 B가 동시에 일어날 확률을 나타냅니다.

**직관적 이해**:
조건부확률은 정보 갱신의 관점에서 이해할 수 있습니다. 사건 B가 발생했다는 새로운 정보가 주어졌을 때, 사건 A의 확률을 갱신하는 것입니다.

예: 한 가방에 빨간 공 3개와 파란 공 2개가 있습니다. 무작위로 한 개의 공을 꺼냈을 때 그것이 빨간색이라는 조건 하에, 처음에 가방에 있던 공이 빨간색일 확률은?

P(첫 번째 공이 빨강|꺼낸 공이 빨강) = P(첫 번째 공이 빨강 ∩ 꺼낸 공이 빨강) / P(꺼낸 공이 빨강)

여기서 "첫 번째 공이 빨강"과 "꺼낸 공이 빨강"은 사실 같은 사건이므로,
P(첫 번째 공이 빨강|꺼낸 공이 빨강) = P(첫 번째 공이 빨강) / P(첫 번째 공이 빨강) = 1

**조건부확률과 독립성**:
두 사건 A와 B가 독립이라면, P(A|B) = P(A)가 성립합니다. 즉, B의 발생 여부가 A의 확률에 영향을 주지 않습니다.

반대로, P(A|B) ≠ P(A)라면 두 사건은 종속(dependent)입니다.

**조건부확률의 성질**:
1. 0 ≤ P(A|B) ≤ 1
2. P(Ω|B) = 1
3. P(∅|B) = 0
4. P(A∪C|B) = P(A|B) + P(C|B) - P(A∩C|B)

**순차적 정보 갱신**:
여러 조건이 순차적으로 주어질 때, 조건부확률을 연속적으로 적용할 수 있습니다:
P(A|B∩C) = P(A∩B∩C) / P(B∩C)

**실생활 응용**:
1. **의학 진단**: 증상이 있을 때 특정 질병일 확률
2. **보험 계리**: 특정 조건을 가진 사람의 위험도 평가
3. **법적 증거 분석**: 증거가 있을 때 피고인이 유죄일 확률
4. **스팸 필터링**: 특정 단어가 포함된 메일이 스팸일 확률

조건부확률은 불확실성 하에서 의사결정을 내리는 강력한 도구이며, 베이즈 정리의 기초가 됩니다.

## 9. 전체확률

**정의**:
B₁, B₂, ..., Bₙ이 다음 조건을 만족하는 표본공간의 분할이라고 합시다:
1. 모든 i ≠ j에 대해 Bᵢ ∩ Bⱼ = ∅ (서로소)
2. B₁ ∪ B₂ ∪ ... ∪ Bₙ = Ω (전체 표본공간 커버)
3. 모든 i에 대해 P(Bᵢ) > 0
